## Anleitung Restore-Drill

1. Letztes verschlüsseltes Backup entschlüsseln: `age -d -i private.key backup.age -o test.db`
2. Backup-DB auf Testinstanz kopieren
3. Testinstanz starten mit `zeiterfassung.db = test.db`
4. Admin-Seite öffnen → Integritätsprüfung ausführen
5. Stichprobe: 2–3 Mitarbeiter-Salden manuell nachrechnen
6. Ergebnis unten eintragen
